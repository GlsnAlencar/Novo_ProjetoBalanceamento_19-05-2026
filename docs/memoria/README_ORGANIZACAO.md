# MEMORIA

Documentacao historica do sistema anterior.

Os arquivos Markdown movidos para esta pasta vieram da raiz do projeto e descrevem fluxos, dados, revisoes e guias anteriores associados principalmente a:

- `public/memoria/`
- `data/memoria/`
- `linhas.json`
- `setores.json`
- `FLUXO_TESTE02`
- Drawflow legado
- revisoes, checklists e guias anteriores ao isolamento `cod_12_05`

Nao usar estes documentos como fonte direta para alterar a Reformulacao sem antes validar o escopo.

Mover para `docs/reformulacao/` somente documentos que falem explicitamente do trabalho atual, do escopo `REFORMULACAO_COD_12_05`, dos arquivos `public/reformulacao/*` ou dos dados `data/reformulacao/*`.
